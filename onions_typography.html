Typographical Treatment

Project Description

This piece was created as part of a larger project for Graphic Design I in Fall 2024. It is a typographical treatment of my last name. I started by choosing a font; I really enjoy cartoonish, sketch-like aesthetics, so I thought that a similar-looking font would be a good representation of me. I changed the color of the font itself - which changed the color of the textâ€™s border - and then I used Live Paint to color the insides of each letter.

Since my last name is Onions, I decided that I wanted to represent my font in terms of layers. I copied and pasted my original piece and layered multiple renditions on top of each other to achieve this effect. Each layer is a different color to emphasize this idea of layers - though the first and last layer are both the same to make the piece seem more uniform.

I created two different versions of this layered typographical treatment - one with complementary colors (blue and orange) and another with analogous colors (different shades of blue). Although it was rather simple to make, I really enjoy this piece because I think that it is a good representation of myself as a person.
